System Architecture

Lyceum's architecture combines powerful microservices with sophisticated AI components to deliver transformative learning experiences.

Core Components

Our platform's magical abilities emerge from five integrated components that work in harmony.

Core System

The central orchestration layer that coordinates all aspects of the Lyceum. It manages user sessions, learning paths, and integrates the various specialized services.

Knowledge Graph

A connected network of concepts, representing relationships between ideas, prerequisites, and learning progressions. Forms the foundation of the educational content structure.

Dialogue System

Facilitates Socratic conversations between learners and the system, guiding discovery through targeted questions and responsive interactions.

Content Engine

Generates and adapts learning materials based on the knowledge graph, learner needs, and pedagogical approaches. Creates multimodal content for diverse learning preferences.

Mentor Service

Provides personalized guidance to learners, offering direction, motivation, and targeted assistance based on individual progress, struggles, and learning patterns.

Technical Documentation

The Lyceum system integrates with the existing Goliath Educational Platform, which consists of several microservices working together to create a comprehensive educational experience.

Ptolemy: Knowledge Mapping Service

Manages educational concepts and their relationships, creating a comprehensive knowledge graph.

  • Technologies: Neo4j, Qdrant, GraphQL, Vector Embeddings
  • Status: Implemented, with ongoing data consistency testing
  • Integration: Provides knowledge structure for content generation and learning paths

Gutenberg: Content Generation Service

Creates adaptive educational materials based on concepts in the knowledge graph.

  • Technologies: FastAPI, MongoDB, RAG (Retrieval-Augmented Generation)
  • Status: Implemented, with ongoing enhancements
  • Integration: Generates content for learner interactions and learning paths

Socrates: Learner Interaction System

Provides conversational interfaces and dialogue-based learning experiences.

  • Technologies: Streamlit, LLM-powered dialogue, Conversation templates
  • Status: Implemented, with RAG demo available
  • Integration: Uses knowledge and content for meaningful learning dialogues

Galileo: Learning Path Recommendation

Creates personalized learning journeys based on concept relationships and learner needs.

  • Technologies: Graph Neural Networks (GNN), PyTorch, FastAPI
  • Status: In development, with ML models being trained
  • Integration: Guides learner progression through optimized learning paths

Lyceum: Educational Experience Layer

Brings together all components into a cohesive, philosophically-grounded learning experience.

  • Technologies: AI-powered media generation, Dynamic web interfaces
  • Status: Under active development, with core design implemented
  • Integration: Orchestrates all services to create transformative education
