Key System Components

Knowledge Graph Engine

The knowledge graph is the heart of Lyceum, representing the interconnected nature of human knowledge. It maps concepts, their relationships, prerequisites, and applications.

Our graph engine uses a combination of expert-created structures and machine learning to continuously evolve and refine these knowledge relationships based on:

  • Educational research and subject matter expertise
  • Actual learning patterns and outcomes
  • New discoveries and content

This graph provides the foundation for personalized learning paths and ensures that learners build knowledge in a structured, coherent way.

Dialogue system interaction flow

Dialogue System

Our dialogue system embodies the Socratic method, engaging learners in meaningful conversations that help them discover and understand concepts through questioning and exploration.

Built on large language models and reinforcement learning, the system:

  • Adapts to the learner's current understanding
  • Asks probing questions that encourage critical thinking
  • Provides explanations that connect to the learner's existing knowledge
  • Identifies and addresses misconceptions through gentle correction

The dialogue system is designed to be a supportive guide rather than an authority, encouraging learners to reach their own conclusions.

Learning Path Engine

The learning path engine creates personalized educational journeys for each learner, combining insights from:

  • The knowledge graph's concept relationships
  • The learner's current knowledge and gaps
  • Optimal learning sequence research
  • The learner's goals and preferences

Using reinforcement learning algorithms, the system dynamically adjusts these paths based on learner performance, engagement, and feedback. The result is a constantly optimizing educational experience that balances challenge with achievability.

Content Generation System

Our content generation system creates educational materials dynamically, tailored to each learner's needs. The system can produce:

  • Clear explanations of complex concepts
  • Practice problems and assessments
  • Visual aids and diagrams
  • Summaries and reviews
  • Connections to real-world applications

Content is generated using a combination of template-based approaches and large language models, with quality assurance processes to ensure accuracy and educational value.

Technical Stack

Front-End Technologies

  • Web Interface: React with TypeScript, responsive design for all device sizes
  • Mobile Applications: React Native for cross-platform compatibility
  • Data Visualization: D3.js and Mermaid for knowledge graphs and learning analytics
  • Accessibility: WCAG 2.1 AA compliance with assistive technology support

Back-End Technologies

  • API Layer: GraphQL with Apollo Server for flexible, efficient data exchange
  • Services: Containerized microservices written in Python and TypeScript
  • Knowledge Graph: Neo4j with custom extensions for educational relationship modeling
  • Vector Store: Qdrant for efficient similarity search and concept embeddings
  • Document Store: MongoDB for content and user data

AI & Machine Learning

  • Dialogue Models: Fine-tuned large language models optimized for educational dialogue
  • Recommender System: Hybrid approach combining collaborative filtering and knowledge graph traversal
  • Knowledge Representation: Custom embeddings that capture educational relationships
  • Learning Analytics: Predictive models for identifying learning patterns and potential interventions

Infrastructure

  • Cloud Platform: Kubernetes-orchestrated containers on AWS
  • Scaling: Autoscaling based on demand with resource optimization
  • Monitoring: Prometheus and Grafana for system health and performance
  • Security: Zero-trust architecture with continuous vulnerability scanning
  • Data Protection: End-to-end encryption and compliance with GDPR, FERPA, and other regulations

Deployment Options

Cloud Hosted (SaaS)

Our primary offering is a fully-managed cloud solution where we handle all infrastructure, scaling, security, and updates. This option provides:

  • Rapid deployment with minimal IT resources required
  • Automatic updates with new features and improvements
  • Enterprise-grade security and compliance
  • Scalability to handle any number of users

On-Premises

For organizations with specific security requirements or existing infrastructure investments, we offer an on-premises deployment option that provides:

  • Complete data sovereignty within your network
  • Integration with existing authentication systems
  • Customized security policies
  • Offline capability for environments with limited connectivity

Hybrid Options

We also support hybrid deployments where sensitive components remain on-premises while leveraging cloud resources for computational-intensive tasks like machine learning. This approach provides a balance of security, performance, and manageability.
